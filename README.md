# Клиент-сервер Named Pipe (C#)

Проект для демонстрации взаимодействия через именованный канал.

## Состав
- `Client/` – консольное приложение, подключается к серверу, отправляет 5 сообщений и выводит ответы.
- `Server/` – консольный сервер, ожидает подключения, принимает сообщения и отвечает.

## Запуск
1. Откройте терминал в папке `Server` и выполните: `dotnet run`
2. Откройте второй терминал в папке `Client` и выполните: `dotnet run`

Клиент автоматически подключится, отправит 5 строк и завершится.

## Требования
- .NET 6.0 или новее

## Лицензия
Учебный проект, свободное использование.
